NAME

sandbox-create - create a new sandbox

SYNOPSIS

sandbox create sandbox [-q] [-h]

DESCRIPTION

sandbox-create creates a new sandbox from the base sandbox. The new sandbox will contain a deep copy of your home directory and a shallow copy of the rest of the server's filesystem. Devices will be remounted using mount(8). The new sandbox will be stored in /var/sandboxes/sandbox.

After creating a sandbox, you'll probably want to run commands in it using sandbox-use(1).

OPTIONS

sandbox
Name of a sandbox.
-q, --quiet
Operate quietly.
-h, --help
Show a help message.
